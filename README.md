# AI助手

基于 uni-app x 的智能AI助手应用。

## 功能

- 🤖 AI问答
- 📁 文件分析
- 🎨 AI P图
- 📹 视频通话
- 🔐 用户登录注册

## 技术栈

- uni-app x (Vue 3 + UTS)
- uniCloud 云函数
- 阿里云通义千问

## 运行

1. 用HBuilderX打开项目
2. 配置API Key：`uniCloud-aliyun/cloudfunctions/common/uni-config-center/ai-config/index.js`
3. 部署云函数并创建数据库集合
4. 运行到内置浏览器或微信开发者工具
